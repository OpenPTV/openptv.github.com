6 Truck Race

Martin Müller and Christian Schulte

[ Start Applet | How to enable Applets ]

Clicking on a truck with the left mouse button starts or stops it. The truck can be speeded up by clicking with the middle mouse button or slowed down with the rightmost mouse button.
