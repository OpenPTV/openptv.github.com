Mozart Demo Applications

This document offers a number of demo applets that can be run directly from within their documentation.

To run the demo applets you have to have Mozart installed (which is easy and is described in ``Installation Manual'') and your Web Browser must be Mozart enabled (which is as easy and is described in Appendix B of ``Installation Manual'').
