2 Loop Support

A new loop facility has now been integrated into the Oz language. It was greatly inspired by the Common Lisp loop macro. The language now has one additional loop keyword. The syntax of the loop macro is:

loop ... end

Within the scope of the loop macro, a number of local macros are allowed and described below.
