1 Introduction

Overview

The Inspector is designed to obliterate the barrier between datastructures and their graphical representation, in particular, to make the user feel operating directly with the values without abstractions involved. This is achieved by providing the following features:

  • Fast drawing services

  • Human readable and predictable data layout

  • Interactive and automatic type-directed tree transformations

  • Multiple views of the same data

  • Widely configurable
