6 Building Mozart for Windows

This chapter describes how to build Mozart for Windows. We describe the tools used in the build process and how to obtain them. We explain how to configure and build both the packages Mozart depends upon and Mozart itself. We conclude by a description of how to create an installer, should you want to do so.
