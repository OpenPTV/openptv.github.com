Known Bugs

This documents summarizes known bugs in Mozart. New bugs can be reported by sending email to bugs@mozart-oz.org or via our bug tracking system reachable under http://www.mozart-oz.org/cgi-bin/oz-bugs. This URL is also a good starting point to find out more about already known bugs (fixed and unfixed ones). However it is intended to be used primarily by the developpers and also documents the state of the system currently being under development, so it might list bugs that do not appear in the released system and vice versa.
