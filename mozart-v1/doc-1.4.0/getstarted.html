Mozart Documentation: Getting Started

System

Installation Manual

guides you through the steps required to install the Mozart distribution on your computer. We make available binary, RPM and source distributions. Sources are also available through our anonymous CVS server.

Mozart Demo Applications

gives you a guided tour of a collection of interactive Oz applications. Each application can be invoked directly from the browser simply by clicking. This requires that the Mozart system first be installed on your computer and Installation Manual explains how to do this.

Changes

documents what has changed between DFKI Oz (Oz 2.0) and Mozart (Oz 3.0).

Documentation

How To Read The Documentation

explains the basic conventions used in structuring, organizing, and presenting the documentation. You must read this first to learn how to navigate the Mozart documentation set and interpret what you see.
