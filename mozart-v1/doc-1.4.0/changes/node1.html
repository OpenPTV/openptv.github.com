1 Mozart 1.4.0

Mozart 1.4.0 is both a development and maintenance release. It contains some substantial changes compared to former releases. It provides some new features, but also discards some former features, making it backwards-incompatible in some cases. This chapter lists the differences between Mozart 1.4.0 and Mozart 1.3.2.
